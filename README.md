# Cosmic Library v1

A modern digital library platform for curated software development resources.

## Stack

- Next.js (App Router)
- TypeScript
- Material UI
- React Three Fiber / Drei
- Vercel

## Features

- Product-style landing page
- Resource library
- Dynamic resource pages
- Static generation with `generateStaticParams`
- Downloadable/viewable PDF resources
- Subtle 3D cosmic background

## Goal

Built as a portfolio-quality frontend project focused on clean architecture, reusable components, and modern UI/UX.
